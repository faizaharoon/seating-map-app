# Seating Map App

## Architecture
- React 18 + TypeScript with strict mode.
- SVG-based seat rendering for smooth performance.
- State management via Context + reducer.
- Selection persists in localStorage.
- Memoized seat components for large arenas.

## Running
```bash
pnpm install
pnpm dev
```

## Features
- Click / keyboard accessible seat selection.
- Maximum 8 seats selectable.
- Selection summary with subtotal.
- Works on desktop and mobile.
